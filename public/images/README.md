# CHW Profiles Screenshot

## Required Image
**Filename:** `chw-profiles-screenshot.png`

**Dimensions:** 1200x675 pixels (16:9 aspect ratio)

**Description:** Screenshot of the Community Health Worker Profiles directory page showing the grid of CHW profile cards.

## How to Create the Screenshot

1. Navigate to `/chws/mock-profiles` page in your browser
2. Take a screenshot of the CHW profiles grid
3. Crop/resize to 1200x675 pixels
4. Save as `chw-profiles-screenshot.png` in this directory

## Alternative
If you don't have a screenshot yet, you can use a placeholder image or create one using a design tool like Figma or Canva showing:
- Multiple CHW profile cards in a grid layout
- Profile photos, names, titles
- Certification badges
- Service area information
- Professional details

The image will be displayed on the home page in the "Are You a Community Health Worker?" section.
