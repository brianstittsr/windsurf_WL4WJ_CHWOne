# Form Template Images

This directory contains thumbnail images for form templates in the CHWOne platform.

## Required Images

The following images should be placed in this directory:

- `attendance-basic.png` - Thumbnail for Basic Attendance Sheet template
- `attendance-training.png` - Thumbnail for Training Attendance Sheet template
- `onboarding-chw.png` - Thumbnail for CHW Onboarding Checklist template
- `onboarding-client.png` - Thumbnail for Client Onboarding Form template
- `default-template.png` - Default thumbnail for templates without a specific image

## Image Requirements

- Dimensions: 800x450 pixels (16:9 aspect ratio)
- Format: PNG with transparency where appropriate
- File size: Optimize for web (< 100KB per image)
- Design: Include visual elements that represent the form's purpose

## Placeholder Images

Until custom images are created, you can use placeholder images with the following naming convention:

```
attendance-basic.png
attendance-training.png
onboarding-chw.png
onboarding-client.png
default-template.png
```

These can be generated using any placeholder image service or created with design tools.
