Common Issues

Authentication Not Found Error

If you're seeing "auth/configuration-not-found" error:

  1. Go to the Firebase Console
  2. Select your project
  3. Go to Authentication (Build → Authentication)
  4. Click "Get started" if you haven't enabled Authentication yet
  5. Enable Email/Password authentication method
  6. Save changes

Firestore Permission Denied

If you're seeing "permission-denied" errors:

  1. Go to the Firebase Console
  2. Select your project
  3. Go to Firestore Database (Build → Firestore Database)
  4. Go to Rules tab
  5. Update rules to allow read/write during development:
rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {
    match /{document=**} {
      // Allow read/write access during development
      allow read, write: if true;
    }
  }
}

Note: These rules are for development only. Use proper security rules in production.
