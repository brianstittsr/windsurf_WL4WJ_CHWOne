# Managing Your Profile

Your profile is your professional identity on CHWOne. Keep it updated to maximize your visibility and effectiveness.

## Accessing Your Profile

1. Click **My Profile** in the sidebar under the Profile section
2. Or click your avatar in the top-right corner and select **Profile**

## Profile Sections

### Basic Information

Edit your fundamental contact details:

| Field | Description | Required |
|-------|-------------|----------|
| First Name | Your given name | Yes |
| Last Name | Your family name | Yes |
| Email | Primary contact email | Yes |
| Phone | Contact phone number | Yes |
| Address | Street address | No |
| City | City of residence | No |
| State | State (defaults to NC) | No |
| Zip Code | Postal code | No |

**How to Edit:**
1. Click the **Edit** button (pencil icon)
2. Modify the fields
3. Click **Save Changes**

### Profile Photo

Add a professional photo to your profile:

1. Click on your current photo or the placeholder
2. Select an image file (JPG, PNG)
3. The image will be automatically compressed
4. Click **Save** to confirm

**Photo Guidelines:**
- Use a clear, professional headshot
- Minimum 200x200 pixels recommended
- Maximum file size: 5MB (auto-compressed)
- Square aspect ratio works best

### Professional Information

Showcase your expertise and background:

| Field | Description |
|-------|-------------|
| Bio | Professional summary (AI enhancement available) |
| Expertise | Areas of specialization |
| Languages | Languages you speak |
| Years of Experience | Time in the field |

**Using AI Bio Enhancement:**
1. Write a basic bio
2. Click **Enhance with AI**
3. Review the enhanced version
4. Accept or modify as needed

### Service Area

Define where you provide services:

| Field | Options |
|-------|---------|
| Region | NC CHWA Regions 1-8 |
| Counties | Multi-select NC counties |
| Organization | Your affiliated organization |
| Role | Your position/title |

### Certification

Track your CHW certification status:

| Field | Description |
|-------|-------------|
| Certification Number | Your official cert number |
| Certification Status | Active, Pending, Expired |
| Expiration Date | When certification expires |

### Privacy Settings

Control who can see your information:

| Setting | Options |
|---------|---------|
| Show Email | Public / Contacts Only / Private |
| Show Phone | Public / Contacts Only / Private |
| Show Address | Public / Contacts Only / Private |
| Include in Directory | Yes / No |

### Social Links

Connect your professional profiles:

- LinkedIn URL
- Twitter/X Handle
- Facebook Page
- Personal Website

## Saving Changes

- Changes are saved when you click **Save Changes**
- A success message confirms the save
- If there's an error, you'll see a red notification

## Profile Visibility

Your profile may appear in:
- **CHW Directory** (if enabled)
- **Organization member lists**
- **Collaboration teams**
- **Referral contacts**

## Tips for a Strong Profile

1. **Complete all sections** - Profiles with more info rank higher
2. **Use a professional photo** - Increases trust and recognition
3. **Keep certification current** - Update before expiration
4. **Write a compelling bio** - Use AI enhancement for polish
5. **List all languages** - Helps match with clients
6. **Update regularly** - Keep information current
