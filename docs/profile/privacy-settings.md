# Privacy Settings

Control how your information is shared and who can see your profile details.

## Accessing Privacy Settings

1. Go to **My Profile**
2. Click the **Privacy** tab
3. Adjust settings as needed
4. Click **Save Changes**

## Contact Visibility Settings

Control who can see your contact information:

### Email Visibility
| Setting | Who Can See |
|---------|-------------|
| **Public** | Anyone on the platform |
| **Contacts Only** | People you've connected with |
| **Private** | Only you and admins |

### Phone Visibility
| Setting | Who Can See |
|---------|-------------|
| **Public** | Anyone on the platform |
| **Contacts Only** | People you've connected with |
| **Private** | Only you and admins |

### Address Visibility
| Setting | Who Can See |
|---------|-------------|
| **Public** | Anyone on the platform |
| **Contacts Only** | People you've connected with |
| **Private** | Only you and admins |

## Directory Inclusion

### Include in CHW Directory
When enabled:
- Your profile appears in the public CHW Directory
- Organizations can find and contact you
- Your expertise and service area are searchable

When disabled:
- Your profile is hidden from directory searches
- Only direct links to your profile work
- Organizations cannot discover you through search

## Data Sharing Preferences

### Share with Organizations
Control what organizations you work with can see:
- Basic profile information
- Certification status
- Service area
- Activity metrics

### Share with Partners
Control data sharing with partner organizations:
- Anonymized statistics only
- Full profile sharing
- No sharing

## Communication Preferences

### Email Notifications
| Type | Description |
|------|-------------|
| **Certification Reminders** | Expiration alerts |
| **Platform Updates** | New features and changes |
| **Referral Notifications** | New referrals assigned |
| **Collaboration Invites** | Project invitations |

### Notification Frequency
- **Immediate** - As events happen
- **Daily Digest** - Once per day summary
- **Weekly Digest** - Once per week summary
- **None** - No email notifications

## Account Security

### Two-Factor Authentication
Enable additional security:
1. Go to Settings > Security
2. Click **Enable 2FA**
3. Scan QR code with authenticator app
4. Enter verification code
5. Save backup codes

### Session Management
View and manage active sessions:
- See all logged-in devices
- Revoke access to specific sessions
- Set session timeout duration

## Data Export

Request a copy of your data:
1. Go to Settings > Privacy
2. Click **Export My Data**
3. Select data types to include
4. Receive download link via email

Exportable data includes:
- Profile information
- Activity history
- Forms submitted
- Referral records

## Account Deletion

To delete your account:
1. Go to Settings > Privacy
2. Click **Delete Account**
3. Confirm your decision
4. Enter password to verify
5. Account will be scheduled for deletion

**Note:** Account deletion is permanent and cannot be undone. Your data will be removed within 30 days.

## Privacy Best Practices

1. **Review settings regularly** - Check quarterly
2. **Use appropriate visibility** - Match your comfort level
3. **Enable 2FA** - Adds security layer
4. **Monitor sessions** - Log out unused devices
5. **Update contact info** - Keep current for notifications
