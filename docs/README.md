## Platform Overview

CHWOne is a comprehensive platform designed for Community Health Workers (CHWs), nonprofit organizations, CHW associations, and state agencies. The platform provides tools for:

- **Profile Management** - Maintain professional profiles with certification tracking
- **Data Collection** - Create and manage health assessment forms
- **Reporting** - Generate insights from collected data
- **Collaboration** - Work together on grants and projects
- **Resource Access** - Find healthcare resources and community services

## User Roles

| Role | Description | Access Level |
|------|-------------|--------------|
| **CHW** | Community Health Worker | Profile, Forms, Referrals, Resources |
| **Nonprofit Staff** | Organization employees | Full platform access |
| **Admin** | Platform administrators | Full access + admin tools |
