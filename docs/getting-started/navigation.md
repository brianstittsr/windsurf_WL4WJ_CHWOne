# Navigation Overview

Learn how to navigate the CHWOne platform efficiently.

## Main Navigation Structure

The platform uses a sidebar navigation system with collapsible sections.

### Sidebar Sections

```
Profile
├── My Profile
├── Settings
└── Manage Profiles

Work
├── Dashboard
├── Referrals
├── Projects
├── Grants
├── Forms
├── Datasets
├── Reports
├── Resources
├── AI Assistant
└── Data Tools

Collaborations
├── My Collaborations
├── Regional Pages
├── CHW Directory
└── NC Legislature

Admin (Admin users only)
├── Platform Settings
├── Users
├── States
├── CHW Associations
├── Nonprofits
├── Analytics
└── Security
```

## Sidebar Controls

### Collapse/Expand
- Click the **Collapse Sidebar** button at the bottom to minimize
- Hover over icons to see tooltips when collapsed
- Click **Expand Sidebar** to restore full view

### Section Toggles
- Click section headers to expand/collapse groups
- Sections remember their state between sessions

## Top Header Bar

The header bar provides:

| Element | Function |
|---------|----------|
| **Page Title** | Shows current page name |
| **Notifications** | Bell icon for alerts |
| **User Menu** | Profile dropdown with quick actions |

### User Menu Options
- View Profile
- Settings
- Sign Out

## Mobile Navigation

On mobile devices:
1. Tap the **hamburger menu** (☰) to open navigation
2. Tap any menu item to navigate
3. Tap **X** or outside to close

## Keyboard Shortcuts

| Shortcut | Action |
|----------|--------|
| `Alt + D` | Go to Dashboard |
| `Alt + P` | Go to Profile |
| `Alt + F` | Go to Forms |
| `Esc` | Close modals/dialogs |

## Breadcrumbs

Some pages show breadcrumb navigation:

```
Dashboard > Projects > Project Name
```

Click any breadcrumb to navigate back.

## Quick Actions

Many pages include quick action buttons:
- **Primary actions** (blue) - Main tasks
- **Secondary actions** (outlined) - Alternative options
- **Danger actions** (red) - Destructive operations

## Search

Use the search functionality to find:
- Forms and templates
- Datasets
- Reports
- Resources
- CHW profiles (in directory)

## Tips for Efficient Navigation

1. **Use the sidebar** - Most features are one click away
2. **Bookmark pages** - Save frequently used pages
3. **Learn shortcuts** - Keyboard navigation is faster
4. **Collapse sections** - Hide unused menu groups
5. **Use breadcrumbs** - Navigate back without sidebar
