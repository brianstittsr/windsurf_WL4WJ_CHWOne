# Resource Directory

Find healthcare resources and community services.

## Overview

The Resource Directory provides access to:
- Healthcare providers
- Social services
- Community organizations
- Support programs
- Educational resources

## Accessing Resources

Navigate to **Work > Resources** in the sidebar.

## Browsing Resources

### Categories

Resources are organized by category:

| Category | Examples |
|----------|----------|
| **Healthcare** | Clinics, hospitals, specialists |
| **Mental Health** | Counseling, therapy, crisis services |
| **Social Services** | Case management, advocacy |
| **Housing** | Shelters, housing assistance |
| **Food** | Food banks, meal programs |
| **Employment** | Job training, placement services |
| **Education** | Adult education, GED programs |
| **Legal** | Legal aid, immigration services |
| **Transportation** | Medical transport, bus passes |
| **Childcare** | Daycare, after-school programs |

### Filtering Resources

Narrow your search:
- **Category** - Type of service
- **Location** - County or region
- **Language** - Services in specific languages
- **Availability** - Currently accepting clients
- **Cost** - Free, sliding scale, insurance

## Resource Details

Each resource listing includes:

### Basic Information
- Organization name
- Description
- Address
- Phone number
- Website
- Hours of operation

### Services Offered
- List of services
- Eligibility requirements
- Application process
- Wait times

### Contact Information
- Primary contact
- Email
- Fax
- Emergency contact

## Searching Resources

### Quick Search
1. Enter keywords in search box
2. Press Enter or click Search
3. View matching results

### Advanced Search
1. Click **Advanced Search**
2. Set multiple filters
3. Combine criteria
4. Search

## Saving Resources

### Favorites
Save frequently used resources:
1. Click the star icon on any resource
2. Access favorites from **My Favorites**

### Lists
Create custom resource lists:
1. Click **Create List**
2. Name your list
3. Add resources
4. Share with colleagues

## Sharing Resources

### Share with Client
Send resource information:
1. Open resource
2. Click **Share**
3. Choose method:
   - Email
   - Print
   - Text message
4. Send

### Share with Colleague
1. Open resource
2. Click **Share**
3. Enter colleague's email
4. Add note
5. Send

## Creating Referrals

Refer clients directly:
1. Open resource
2. Click **Create Referral**
3. Enter client information
4. Submit referral

## Regional Resources

### NC CHWA Regions

Resources are organized by NC CHWA regions:
- Region 1 - Western NC
- Region 2 - Northwest Piedmont
- Region 3 - Piedmont Triad
- Region 4 - Triangle
- Region 5 - Sandhills
- Region 6 - Cape Fear
- Region 7 - Eastern NC
- Region 8 - Northeast NC

### Viewing Regional Resources
1. Select your region
2. Browse local resources
3. Filter by category

## Suggesting Resources

Add new resources to the directory:

1. Click **Suggest Resource**
2. Enter organization information:
   - Name
   - Address
   - Contact info
   - Services
3. Submit for review
4. Admin will verify and add

## Resource Updates

### Reporting Issues
If information is incorrect:
1. Open resource
2. Click **Report Issue**
3. Describe the problem
4. Submit

### Requesting Updates
To update resource information:
1. Open resource
2. Click **Request Update**
3. Enter new information
4. Submit for review

## Best Practices

1. **Verify before referring** - Call to confirm availability
2. **Check eligibility** - Ensure client qualifies
3. **Note hours** - Confirm operating hours
4. **Save favorites** - Quick access to common resources
5. **Update regularly** - Report outdated information

## Troubleshooting

### Resource not found?
- Try different keywords
- Expand search area
- Check category filters
- Suggest new resource

### Information outdated?
- Report the issue
- Request update
- Contact admin

### Can't create referral?
- Verify you're logged in
- Check resource accepts referrals
- Try different resource
