# Referrals Management

Track client referrals and coordinate care transitions.

## Overview

The Referrals tool allows you to:
- Create and track client referrals
- Coordinate with service providers
- Monitor referral outcomes
- Generate referral reports

## Accessing Referrals

Navigate to **Work > Referrals** in the sidebar.

## Understanding Referrals

A referral connects a client with services they need:

```
Client Need → Referral Created → Service Provider → Service Delivered → Outcome Tracked
```

## Creating Referrals

### Step 1: Start New Referral

1. Click **Create Referral**
2. Enter client information:
   - Client name/ID
   - Contact information
   - Current situation

### Step 2: Specify Service Need

Select the type of service needed:
- Healthcare services
- Social services
- Housing assistance
- Food assistance
- Mental health
- Substance abuse
- Employment
- Education
- Legal services
- Other

### Step 3: Select Provider

Choose a service provider:
1. Browse available providers
2. Filter by service type and location
3. View provider details
4. Select provider

### Step 4: Add Details

Include additional information:
- Urgency level
- Special requirements
- Notes for provider
- Attachments

### Step 5: Submit Referral

1. Review all information
2. Click **Submit Referral**
3. Provider is notified

## Referral Status

| Status | Description |
|--------|-------------|
| **Draft** | Being prepared |
| **Submitted** | Sent to provider |
| **Accepted** | Provider accepted |
| **In Progress** | Service being delivered |
| **Completed** | Service delivered |
| **Declined** | Provider declined |
| **Cancelled** | Referral cancelled |

## Tracking Referrals

### Referral List

View all your referrals:
- Filter by status
- Sort by date
- Search by client

### Referral Details

Click any referral to view:
- Client information
- Service details
- Provider information
- Status history
- Communications
- Outcomes

## Referral Actions

| Action | Description |
|--------|-------------|
| **View** | See full details |
| **Edit** | Modify referral |
| **Follow Up** | Add update |
| **Close** | Mark complete |
| **Cancel** | Cancel referral |

## Communication

### Adding Notes

Document interactions:
1. Open referral
2. Click **Add Note**
3. Enter update
4. Save

### Contacting Provider

Reach out to service provider:
1. Open referral
2. Click **Contact Provider**
3. Send message
4. Message logged in referral

## Outcomes Tracking

Record referral outcomes:

### Outcome Types
| Outcome | Description |
|---------|-------------|
| **Successful** | Client received service |
| **Partial** | Some services received |
| **Unsuccessful** | Service not received |
| **Client Declined** | Client chose not to proceed |
| **Unable to Contact** | Lost contact with client |

### Recording Outcomes

1. Open completed referral
2. Click **Record Outcome**
3. Select outcome type
4. Add notes
5. Save

## Referral Reports

Generate referral analytics:

### Available Reports
| Report | Content |
|--------|---------|
| **Summary** | Overview of all referrals |
| **By Service** | Breakdown by service type |
| **By Provider** | Provider performance |
| **Outcomes** | Success rates |
| **Timeline** | Processing times |

### Creating Reports

1. Click **Reports**
2. Select report type
3. Choose date range
4. Generate report
5. Export if needed

## Service Provider Directory

Find service providers:

1. Click **Provider Directory**
2. Search or browse
3. Filter by:
   - Service type
   - Location
   - Availability
   - Languages
4. View provider details
5. Create referral directly

## Best Practices

1. **Complete information** - Include all relevant details
2. **Timely follow-up** - Check status regularly
3. **Document everything** - Keep thorough notes
4. **Track outcomes** - Record results
5. **Build relationships** - Maintain provider connections

## Privacy Considerations

When creating referrals:
- Only share necessary information
- Obtain client consent
- Follow HIPAA guidelines
- Protect sensitive data

## Troubleshooting

### Referral won't submit?
- Check all required fields
- Verify provider is available
- Try refreshing page

### Provider not responding?
- Send follow-up message
- Try alternative contact
- Consider different provider

### Can't find provider?
- Expand search area
- Try different service category
- Contact admin to add provider
