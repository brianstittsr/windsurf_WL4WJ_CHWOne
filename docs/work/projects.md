# Projects Management

Coordinate and track health initiatives and community programs.

## Overview

The Projects tool allows you to:
- Create and manage health projects
- Assign team members
- Track tasks and milestones
- Monitor progress
- Collaborate with partners

## Accessing Projects

Navigate to **Work > Projects** in the sidebar.

## Creating Projects

### Step 1: Start New Project

1. Click **Create Project**
2. Enter project details:
   - Project name
   - Description
   - Start date
   - End date
   - Category

### Step 2: Configure Project

Set up project structure:
- Define objectives
- Set milestones
- Assign team members
- Allocate resources

### Step 3: Launch Project

1. Review all settings
2. Click **Create Project**
3. Project appears in your list

## Task Management

### Creating Tasks

1. Open project
2. Go to **Tasks** tab
3. Click **Add Task**
4. Enter:
   - Task name
   - Description
   - Assignee
   - Due date
   - Priority
5. Save

### Task Properties

| Property | Options |
|----------|---------|
| **Priority** | Low, Medium, High, Urgent |
| **Status** | To Do, In Progress, Review, Done |
| **Assignee** | Team member |
| **Due Date** | Target completion |

### Task Views

| View | Description |
|------|-------------|
| **List** | Traditional task list |
| **Board** | Kanban-style columns |
| **Calendar** | Timeline view |
| **Gantt** | Project timeline |

## Team Management

### Adding Team Members

1. Open project
2. Go to **Team** tab
3. Click **Add Member**
4. Search for user or enter email
5. Assign role
6. Send invitation

### Team Roles

| Role | Permissions |
|------|-------------|
| **Owner** | Full control |
| **Manager** | Edit project, manage team |
| **Member** | Complete tasks, add comments |
| **Viewer** | View only |

### Status Updates

Post regular updates:
1. Open project
2. Click **Add Update**
3. Write status summary
4. Attach files if needed
5. Post update

## Linking to Grants

Connect projects to grants:

1. Open project
2. Go to **Settings**
3. Click **Link Grant**
4. Select associated grant
5. Save

Benefits:
- Shared milestones
- Unified reporting
- Budget alignment

## Project Reports

Generate project reports:

1. Open project
2. Click **Reports**
3. Select report type:
   - Progress summary
   - Task completion
   - Team activity
   - Timeline status
4. Generate and download

## Best Practices

1. **Clear objectives** - Define success criteria
2. **Regular updates** - Post weekly status
3. **Task breakdown** - Keep tasks manageable
4. **Assign owners** - Every task needs an owner
5. **Track blockers** - Address issues quickly

## Troubleshooting

### Can't create project?
- Check you have permission
- Verify all required fields
- Try refreshing page

### Team member can't access?
- Verify invitation was sent
- Check their email
- Resend invitation

### Tasks not syncing?
- Refresh the page
- Check internet connection
- Clear browser cache
