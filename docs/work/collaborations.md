# Collaborations

Work together with partner organizations on grants and projects.

## Overview

The Collaborations tool allows you to:
- View grant collaborations
- Track partner organizations
- Monitor project progress
- Coordinate activities

## Accessing Collaborations

Navigate to **Collaborations > My Collaborations** in the sidebar.

### Overview Card

Each collaboration shows:

## Viewing Collaborations

### Collaboration List

The main page shows all your collaborations:
- Filter by status
- Sort by date or progress
- Search by name

### Collaboration Details

Click **View Collaboration** to see:
- Full grant information
- All partner details
- Milestones and tasks
- Documents and reports
- Activity timeline

## Collaboration Status

| Status | Description |
|--------|-------------|
| **Active** | Currently running |
| **Pending** | Awaiting approval |
| **Completed** | Successfully finished |
| **Cancelled** | Terminated |

## Working with Partners

### Partner Information

View partner details:
- Organization name
- Role in collaboration
- Contact information
- Responsibilities

### Communication

Coordinate with partners:
- Shared activity feed
- Document sharing
- Task assignments
- Status updates

## Milestones

Track collaboration milestones:

### Viewing Milestones
1. Open collaboration
2. Go to **Milestones** tab
3. View all milestones
4. Check status and dates

### Milestone Status
| Status | Meaning |
|--------|---------|
| **Pending** | Not started |
| **In Progress** | Currently working |
| **Completed** | Finished |
| **Overdue** | Past deadline |

## Documents

Access collaboration documents:

### Document Types
- Grant agreements
- Work plans
- Reports
- Meeting notes
- Deliverables

### Sharing Documents
1. Open collaboration
2. Go to **Documents** tab
3. Click **Upload**
4. Select file
5. Set permissions

## Reporting

Generate collaboration reports:

### Report Types
| Report | Content |
|--------|---------|
| **Progress** | Overall status |
| **Partner Activity** | Organization contributions |
| **Milestones** | Deliverable status |
| **Financial** | Budget utilization |

### Creating Reports
1. Open collaboration
2. Click **Reports**
3. Select report type
4. Generate and download

## Timeline View

## Notifications

Stay informed about:
- Status changes

## Best Practices

1. **Regular check-ins** - Review progress weekly
2. **Clear communication** - Keep partners informed
3. **Document everything** - Maintain records
4. **Track milestones** - Monitor deadlines
5. **Celebrate wins** - Acknowledge achievements

## Troubleshooting

### Can't see collaboration?
- Check you're a member
- Verify permissions
- Contact lead organization

### Partner not responding?
- Try alternative contact
- Escalate to lead
- Document attempts

### Progress not updating?
- Refresh the page
- Check milestone status
- Contact admin
