# Grants Management

Track funding opportunities, manage applications, and monitor grant progress.

## Overview

The Grants tool allows you to:
- Discover funding opportunities
- Track grant applications
- Manage awarded grants
- Monitor milestones and deliverables
- Collaborate with partner organizations

## Accessing Grants

Navigate to **Work > Grants** in the sidebar.

## Grant Lifecycle

```
Discovery → Application → Review → Award → Implementation → Reporting → Closeout
```

### 1. Discovery
Find relevant funding opportunities:
- Browse available grants
- Filter by category, amount, deadline
- Save interesting opportunities

### 2. Application
Prepare and submit applications:
- Upload grant documents
- Use AI to analyze requirements
- Track application status

### 3. Review
Monitor application progress:
- View review status
- Respond to requests for information
- Track decision timeline

### 4. Award
Manage awarded grants:
- Accept award terms
- Set up project structure
- Assign team members

### 5. Implementation
Execute grant activities:
- Track milestones
- Manage budget
- Document activities

### 6. Reporting
Submit required reports:
- Progress reports
- Financial reports
- Outcome data

### 7. Closeout
Complete grant requirements:
- Final reporting
- Documentation
- Lessons learned

## Grant Analyzer Wizard

Use AI to analyze grant documents:

### Step 1: Upload Document
1. Click **Create Grant** or **Analyze Grant**
2. Upload your grant document (PDF, Word)
3. Wait for AI analysis

### Step 2: Review Extracted Information
The AI extracts:
- Grant title and description
- Funding amount
- Timeline and deadlines
- Requirements and deliverables
- Reporting obligations

### Step 3: Configure Project
Set up your grant project:
- Add collaborating organizations
- Define roles and responsibilities
- Set up data collection plan
- Create reporting schedule

### Step 4: Generate Forms
Auto-create data collection forms:
- Based on grant requirements
- Customizable templates
- Linked to reporting needs

## Grant Status Types

| Status | Description |
|--------|-------------|
| **Draft** | Being prepared |
| **Submitted** | Application sent |
| **Under Review** | Being evaluated |
| **Awarded** | Funding approved |
| **Active** | Currently running |
| **Completed** | Successfully finished |
| **Declined** | Not funded |
| **Cancelled** | Withdrawn |

## Managing Collaborating Entities

Add organizations working on the grant:

### Entity Roles
| Role | Responsibilities |
|------|------------------|
| **Lead** | Primary applicant, overall management |
| **Partner** | Co-implementer, shared activities |
| **Evaluator** | Assessment and evaluation |
| **Stakeholder** | Advisory, community input |

### Adding Entities
1. Open grant details
2. Go to **Collaborators** tab
3. Click **Add Organization**
4. Enter organization details
5. Assign role and responsibilities
6. Save

## Milestone Tracking

Track grant progress with milestones:

### Creating Milestones
1. Open grant details
2. Go to **Milestones** tab
3. Click **Add Milestone**
4. Enter:
   - Milestone name
   - Due date
   - Description
   - Deliverables
5. Save

### Milestone Status
| Status | Meaning |
|--------|---------|
| **Pending** | Not yet started |
| **In Progress** | Currently working |
| **Completed** | Successfully finished |
| **Overdue** | Past due date |
| **At Risk** | May not meet deadline |

## Best Practices

1. **Start early** - Begin applications well before deadlines
2. **Document everything** - Keep thorough records
3. **Track milestones** - Monitor progress regularly
4. **Communicate** - Keep collaborators informed
5. **Report on time** - Never miss reporting deadlines

## Troubleshooting

### Document upload fails?
- Check file format (PDF, Word)
- Verify file size (max 25MB)
- Try re-saving the document

### AI analysis incomplete?
- Ensure document is text-based (not scanned image)
- Try uploading a cleaner version
- Manually add missing information

### Collaborator can't access?
- Verify their email address
- Check permission settings
- Resend invitation
