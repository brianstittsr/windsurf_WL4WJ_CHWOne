## Overview

The CHWOne Grant Wizard is an AI-powered tool that streamlines the process of setting up grant collaboration projects. It automatically extracts information from grant documents, identifies collaborating entities, defines data collection requirements, and creates project milestones.

### Key Features

- **AI-Powered Document Analysis**: Upload a PDF grant document and let AI extract all relevant information
- **Entity Management**: Track all collaborating organizations and their roles
- **Data Collection Planning**: Define methods, frequencies, and responsible parties
- **Milestone Tracking**: Create project timelines with dependencies
- **Form Generation**: Automatically create data collection forms
- **Dashboard Generation**: Build real-time reporting dashboards
- **Collaboration View**: Unified view of all grant partnerships

   - Grant title
   - Description
   - Funding source
   - Budget amount
   - Start and end dates

### Step 2: Entity Details

**Purpose:** Identify and manage collaborating organizations.

**Actions:**
1. Review AI-extracted entities
2. Add/edit collaborating organizations
3. Assign roles (Lead, Partner, Evaluator, Stakeholder)
4. Add contact information
5. Define responsibilities

**Entity Roles:**
- **Lead**: Primary organization responsible for grant management
- **Partner**: Organizations actively participating in project delivery
- **Evaluator**: Organizations responsible for assessment and evaluation
- **Stakeholder**: Organizations with interest but limited active participation

### Generate AI Dashboard

From the Grant Management page, click "Generate Dashboard" on any grant to:

1. **Analyze Reporting Requirements**: AI reviews milestones and forms
2. **Generate Dashboard Components**:
   - KPI Metrics (Participants, Budget, Milestones, Reports)
   - Charts (Line, Pie, Bar)
   - Data Tables
3. **Preview Dashboard**: See all components
4. **Save to Grant**: Dashboard config is stored with the grant

## Summary

The Grant Wizard and Collaborations integration provides a complete workflow for:

1. **Document Upload & AI Analysis** - Extract grant information automatically
2. **Entity Management** - Track all collaborating organizations
3. **Data Collection Planning** - Define what data to collect and how
4. **Project Planning** - Create milestones and timelines
5. **Form Generation** - Create data collection forms automatically
6. **Dashboard Generation** - Build real-time reporting dashboards
7. **Collaboration Tracking** - Monitor progress across all partners

This integrated system ensures that grant collaboration projects are set up correctly from the start, with all necessary tracking and reporting infrastructure in place.
