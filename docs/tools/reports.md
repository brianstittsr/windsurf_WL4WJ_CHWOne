# Reports

Generate insights and visualizations from your data.

## Overview

The Reports tool allows you to:
- Create data-driven reports
- Generate visualizations and charts
- Share reports with stakeholders
- Export reports as PDF

## Accessing Reports

Navigate to **Work > Reports** in the sidebar.

## Report Tabs

### All Reports
View all reports you have access to.

### My Reports
View reports you've created.

### Shared with Me
View reports others have shared with you.

## Creating Reports

### Step 1: Start Report Creation

1. Click **Create Report**
2. The AI-powered report builder opens

### Step 2: Describe Your Report

Use natural language to describe what you need:

```
Example prompts:
- "Create a monthly summary of client visits by county"
- "Show diabetes screening results with trends over time"
- "Generate a grant progress report with milestones"
```

### Step 3: Select Data Sources

Choose datasets to include:
1. Browse available datasets
2. Select one or more datasets
3. The AI will suggest relevant visualizations

### Step 4: Configure Sections

Reports can include multiple sections:

| Section Type | Description |
|--------------|-------------|
| **Text** | Written content, summaries |
| **Visualization** | Charts and graphs |
| **Summary** | Key findings and insights |
| **Table** | Data tables |

### Step 5: Generate Report

1. Review the report configuration
2. Click **Generate Report**
3. Wait for processing
4. Review the generated report

## Visualization Types

| Type | Best For |
|------|----------|
| **Bar Chart** | Comparing categories |
| **Line Chart** | Trends over time |
| **Pie Chart** | Part-to-whole relationships |
| **Area Chart** | Cumulative trends |
| **Scatter Plot** | Correlations |
| **Table** | Detailed data display |

## Report Actions

| Action | Description |
|--------|-------------|
| **View** | Open full report |
| **Edit** | Modify report content |
| **Share** | Send to others |
| **Download PDF** | Export as PDF file |
| **Delete** | Remove report |

## Report Status

| Status | Meaning |
|--------|---------|
| **Draft** | Work in progress |
| **Generating** | Being processed |
| **Complete** | Ready to view |
| **Error** | Generation failed |

## Viewing Reports

Click on any report to view:

### Report Header
- Title and description
- Created/updated dates
- Author information

### Report Body
- Text sections
- Visualizations
- Data tables
- Key findings

### Report Actions
- Edit button
- Share button
- Download PDF button

## Editing Reports

Modify existing reports:

1. Open the report
2. Click **Edit**
3. Make changes:
   - Update text content
   - Modify visualizations
   - Add/remove sections
4. Click **Save Changes**

## Sharing Reports

Share reports with others:

### Share with Users
1. Click **Share** on a report
2. Enter email addresses
3. Set permission level:
   - **View** - Can only view
   - **Comment** - Can add comments
   - **Edit** - Can modify
4. Click **Send**

### Get Shareable Link
1. Click **Share**
2. Toggle **Anyone with link**
3. Copy the link
4. Share via email or message

## Exporting Reports

### Download as PDF
1. Open the report
2. Click the **PDF** icon
3. Wait for PDF generation
4. Download the file

### Export Data
1. Open the report
2. Click **Export Data**
3. Choose format (CSV, Excel)
4. Download the file

## Scheduling Reports

Set up automatic report generation:

1. Open report settings
2. Click **Schedule**
3. Configure:
   - Frequency (daily, weekly, monthly)
   - Recipients
   - Delivery time
4. Enable scheduling

## Best Practices

1. **Clear titles** - Describe report content
2. **Appropriate visualizations** - Match chart to data
3. **Key findings first** - Lead with insights
4. **Regular updates** - Keep data current
5. **Share appropriately** - Consider audience

## Troubleshooting

### Report won't generate?
- Check dataset availability
- Verify data has required columns
- Try simpler visualization types

### Visualizations look wrong?
- Check data types (numbers vs text)
- Verify date formats
- Ensure sufficient data points

### PDF export fails?
- Try smaller report
- Check browser popup blocker
- Wait for report to fully load
